Questao01: Quantas iterações são necessárias para encontrar os pesos corretos da base de treinamento?

	Selecionando as linhas da tabela sequencialmente são necessárias 2 iterações apenas, mas selecionando aleatoriamente foi possível encontrar soluções com até 8 iterações.

Questao03: O programa parece ter alguma inteligencia? Como ele se comporta?

	O programa possui inteligencia, pois é capaz de aprender. Para esse caso ele possui poucos dados para treinar os pesos, mas com uma base de dados extensa e bem direcionada para o problema que se quer resolver ele pode aprender e ser capaz de prever resultados futuros.
